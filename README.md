# agentteam
Python monorepo containing agent teams managed via crewai.

[![Gatekeeper](https://github.com/matze298/agentteam/actions/workflows/gatekeeper.yml/badge.svg)](https://github.com/matze298/agentteam/actions/workflows/gatekeeper.yml)

## Setup
* Prerequisite: Install `uv`
* Run `./setup.py`


## Security Considerations
* Only run in Docker container (ensure Docker is running!)
* Directory Sandboxing
* Strict HitL setup
